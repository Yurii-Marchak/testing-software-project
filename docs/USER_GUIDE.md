# Керівництво користувача

## 1. Призначення

Цей документ описує, як користуватися системою `PC Build Manager` без заглиблення в код.

## 2. Вхід до системи

1. Запустіть сервер:

```powershell
py main.py
```

2. Відкрийте у браузері:

```text
http://127.0.0.1:5000/
```

## 3. Головні розділи

У лівому меню доступні:
- `Дашборд`
- `Замовлення`
- `Клієнти`
- `Збірки ПК`
- `Комплектуючі`

## 4. Робота з клієнтами

### Створення клієнта

1. Перейдіть у розділ `Клієнти`.
2. Натисніть `Новий клієнт`.
3. Заповніть:
   - прізвище
   - ім’я
   - дату народження
   - телефон
   - email
4. Натисніть `Зареєструвати клієнта`.

### Вимоги до даних

- телефон: `+380XXXXXXXXX`
- email: `name@example.com`

### Редагування клієнта

1. Знайдіть рядок клієнта в таблиці.
2. Натисніть `Редагувати`.
3. Оновіть дані.
4. Натисніть `Зберегти зміни`.

### Видалення клієнта

1. Знайдіть клієнта в таблиці.
2. Натисніть `Видалити`.

Примітка: якщо клієнт уже використовується в замовленнях, система може заблокувати видалення.

## 5. Робота зі збірками ПК

### Створення збірки

1. Перейдіть у `Збірки ПК`.
2. Натисніть `Нова збірка`.
3. Оберіть:
   - GPU
   - CPU
   - материнську плату
   - RAM
   - PSU
   - корпус
4. Введіть тип збірки.
5. Перевірте preview.
6. Натисніть `Створити збірку`.

### Редагування збірки

1. У таблиці збірок натисніть `Редагувати`.
2. Оновіть конфігурацію.
3. Натисніть `Зберегти зміни`.

### Видалення збірки

1. У таблиці натисніть `Видалити`.

## 6. Робота із замовленнями

### Створення замовлення

1. Перейдіть у `Замовлення`.
2. Натисніть `Нове замовлення`.
3. Оберіть клієнта.
4. Оберіть збірку ПК.
5. Вкажіть дату та час завершення.
6. Виберіть:
   - статус оплати
   - статус замовлення
7. Перевірте preview.
8. Натисніть `Оформити замовлення`.

### Пошук клієнта за телефоном

У верхній таблиці сторінки `Замовлення` можна відфільтрувати клієнтів за номером телефону.

### Редагування замовлення

1. У таблиці замовлень натисніть `Редагувати`.
2. Змініть потрібні поля.
3. Збережіть зміни.

### Видалення замовлення

1. У таблиці замовлень натисніть `Видалити`.

### Чек останньої операції

Після створення замовлення сторінка може показати чек:
- клієнт
- склад збірки
- загальна ціна
- сума до сплати

## 7. Робота з комплектуючими

### Створення нового компонента

1. Перейдіть у `Комплектуючі`.
2. Виберіть секцію:
   - GPU
   - CPU
   - Motherboard
   - RAM
   - PSU
   - Case
3. Натисніть `Додати`.
4. Заповніть форму.
5. Натисніть `Створити запис`.

### Редагування компонента

1. У потрібній секції знайдіть запис.
2. Натисніть `Редагувати`.
3. Внесіть зміни.
4. Натисніть `Зберегти зміни`.

### Видалення компонента

1. Натисніть `Видалити` в рядку компонента.

## 8. Дашборд

На дашборді доступні:
- кількість замовлень;
- кількість клієнтів;
- кількість збірок;
- кількість неоплачених замовлень;
- список останніх замовлень.

## 9. Поведінка при помилках

### Помилки форми

Якщо введені дані некоректні:
- форма не закривається;
- повідомлення показується прямо в модалці;
- уже введені значення не губляться.

### Сторінки помилок

Кастомні сторінки:
- `404` — сторінку не знайдено;
- `500` — внутрішня помилка сервера.

## 10. Корисні поради

- якщо сторінка виглядає застарілою після змін, оновіть її через `Ctrl+F5`;
- якщо застосунок не стартує, перевірте `.env`;
- якщо не працює БД, перевірте `DB_PASSWORD`, `DB_HOST`, `DB_NAME`.
